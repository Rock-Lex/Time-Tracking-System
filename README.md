 ## &ensp; &ensp;  <b> How to install this project? </b>

   - make <br/>
   - cmake <br/>

## &ensp; &ensp;  <b> How to run this project? </b>
 - git clone 
 - open project with Visual Studio
 - clear build folder with clear.sh
 - in build folder:
 &ensp; - cmake ..
 &ensp; - make
 &ensp; - ./Zeiterfassung
